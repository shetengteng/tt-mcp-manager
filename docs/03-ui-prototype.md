- Primary: #3b82f6 (蓝色) - 按钮、链接、强调
- Success: #22c55e (绿色) - 运行中状态
- Warning: #f59e0b (橙色) - 警告信息
- Error: #ef4444 (红色) - 错误状态
- Info: #6366f1 (紫色) - 提示信息

中性色:
- Background: #ffffff (白色)
- Surface: #f8fafc (浅灰)
- Border: #e2e8f0 (边框灰)
- Text Primary: #0f172a (深色文字)
- Text Secondary: #64748b (次要文字)
```

### 字体

```
字体家族:
- 系统默认: -apple-system, BlinkMacSystemFont, "Segoe UI", 
  "PingFang SC", "Microsoft YaHei", sans-serif

字号:
- 标题 H1: 24px (bold)
- 标题 H2: 20px (semibold)
- 标题 H3: 18px (semibold)
- 正文: 14px (regular)
- 小字: 12px (regular)
- 代码: 13px (monospace)
```
